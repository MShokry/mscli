# Usage
<!-- usage -->
```sh-session
$ npm install -g @mshokry/mscli
$ mscli COMMAND
running command...
$ mscli (-v|--version|version)
@mshokry/mscli/0.2.0 darwin-x64 node-v12.16.3
$ mscli --help [COMMAND]
USAGE
  $ mscli COMMAND
...
```
