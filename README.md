# Usage
<!-- usage -->
```sh-session
$ npm install -g mscli
$ mscli COMMAND
running command...
$ mscli (-v|--version|version)
mscli/0.0.0 darwin-x64 node-v12.16.3
$ mscli --help [COMMAND]
USAGE
  $ mscli COMMAND
...
```
